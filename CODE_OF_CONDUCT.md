# Code of Conduct

## Our Commitment

We want this project to be a welcoming space for everyone. Whether you're reporting a bug, suggesting a feature, or contributing code, we value your participation.

## Guidelines

**Be respectful** — Treat others as you'd like to be treated. Disagreements happen, but stay professional.

**Be constructive** — Focus on helping improve the project. Offer solutions, not just criticism.

**Be inclusive** — Welcome newcomers and help them get started. Remember that everyone was a beginner once.

**Be patient** — This is a community project maintained in spare time. Responses may take a few days.

## Unacceptable Behavior

- Personal attacks or insults
- Harassment of any kind
- Spam or off-topic self-promotion
- Publishing others' private information

## Reporting Issues

If you experience or witness unacceptable behavior, please contact us at **info@vibecodingacademy.nl**. We'll review the situation and respond appropriately.

## Scope

This Code of Conduct applies to all project spaces: GitHub issues, pull requests, discussions, and any other communication channels related to this project.
